# Sub-project #5: Garden — Spec

**Parent:** Perfectly Tranquillo. Designs at `docs/design/README.md` § 4 (Garden / Progress) + prototype `docs/design/prototype/src/screens-progress.jsx`.

**Job:** Replace the Garden placeholder with a real progress screen — a glass jar of seashells (one per ritual completion) + a 7-day average bar per Satir resource. Persist Stones to Room so the jar fills up across launches.

## Scope (in)

- **Stone persistence (Room):** new `StoneEntity` (id, kind, awardedAt, source) + `StoneDao` + `StonesRepository`. Bump DB to v4 (destructive — no real users yet). `AppViewModel.addStone` writes through; `AppViewModel.stones` is a `StateFlow` backed by the DAO.
- **Seashell composable:** `ui/components/Seashell.kt` — scalloped fan SVG-equivalent in Compose Canvas. 8 colors (`pearl, coral, rose, butter, sky, lilac, moss, sand`), per-shell tilt + radial gradient + ridges + umbo highlight. Matches prototype `Seashell` exactly.
- **Jar composable:** `screens/garden/JarOfShells.kt` — glass jar (300×320 dp default) drawn with Compose Canvas: body path with shoulders + neck + rim, sand floor ellipse, left/right highlight streaks, rim shine. Stacks shells in 5-column staggered grid bottom-up.
- **Stone→shell color mapping:** `KindToShellColor` — `Moon→pearl, Sand→butter, Jade→moss, Coral→coral, Deep→sky, Shell→sand`.
- **Garden screen:** Header (eyebrow "{N} days tending" / serif title "Your quiet garden") + Card 1 (jar + "shells collected" eyebrow + italic blurb + 5-row legend with counts) + Card 2 ("Resources · 7 day average" + 8 horizontal bars: label / gradient bar / mono percentage).
- **Resource averages computation:** average over the last 7 days of `(am + pm) / 2` per resource from `mandala_entries`. Add `MandalaRepository.fillsByDateInRange(startDate, endDate)` returning `Flow<Map<String, Map<ResourceKey, AmPmFill>>>`.
- **Empty state:** if zero shells, jar still renders but legend counts are all 0; small italic hint "your shells will gather here as you keep your rituals". If <7 days of data, averages compute over what exists.
- Replace `Route.Garden` placeholder with `GardenScreen(viewModel)`.
