# Sub-project #3: Habits ‚Äî Spec

**Parent:** Perfectly Tranquillo. Design at `docs/design/README.md` ¬ß 2 + `docs/design/prototype/src/screens-habits.jsx`.

**Job:** real Habits screen (daily + weekly), persisted to Room. Toggling a habit fills its mandala resource per the resource‚Üîhabit map. The Habits placeholder gets replaced.

## Scope (in)

- Two Room entities: `HabitEntity` (daily) and `WeeklyHabitEntity`. Fields: id, label, hint, doneToday (date-stamp + boolean), streak, remindAt (HH:mm or null). Weekly adds `dayOfWeek: Int` (0..6, Sunday=0).
- Seed default habits on first launch (per design ¬ß 1 / `INITIAL_STATE` in app.jsx): 7 daily habits and 4 weekly habits ‚Äî see `docs/design/prototype/src/app.jsx` `INITIAL_STATE.habits` & `weeklyHabits`. Use exactly those labels + hints + ids + initial streaks.
- DAOs + flow-based reads.
- Extend `AppViewModel` with: `dailyHabits: StateFlow<List<HabitUi>>`, `weeklyHabits`, functions `toggleHabit(id)`, `setHabitReminder(id, time?)`, `addDailyHabit(label, hint)`, `addWeeklyHabit(label, hint, day)`, `removeHabit(id)`.
- HabitsScreen with two tabs (Daily / Weekly) using `TabRow`. Daily tab = list of HabitRow; Weekly tab = day strip + selected-day filtered HabitRows.
- HabitRow component: small `Stone` (Sand for daily, Coral for weekly) as the toggle (inverted color when undone, full color when done), Caveat label, Nunito hint, streak chip "üå± N", optional bell icon ‚Üí opens inline time-picker.
- Toggling a habit recomputes a small `pmFill` bonus on its mapped Satir resource (per ¬ß "Resource‚Üîhabit/action mapping" in `docs/design/README.md` v1 ‚Äî keep that mapping). Apply +0.15 to `resources[mapped].pm` capped at 1.0. Save to a separate `HabitFillEntity` (date-keyed) so the contribution is recoverable across sessions.
- Snapshot pill on Home updates: "X/N habits" reads daily completion.
- Bump versionCode 4, versionName 0.4.0. Tag sub3-complete. Release v0.4.0.
